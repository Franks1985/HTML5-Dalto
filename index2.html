Metadatos

Descripcion de meta
Los metadatos son una especie de informacion que describe la informacion de un objeto, basicamente describe recursos,el metadato se trabaja con la etiqueta < meta>, es la que se pone en el head, en el head esta todo lo que no se puede ver, osea que la etiqueta meta no va a ser vista por el usuario.
para cambiar la codificacion de nuestra pagina, colocamos la etiqueta meta con el atributo charset y le damos la codificacion con UTF-8.
Tipos de etiquetas meta
  • Etiqueta: meta
  • Atributo: name
  • Contenido: =""
meta name="keywords" content="harina, huevo, leche"
con esta etiqueta le indicamos a los buscadores las palabras claves para encontrar nuestra pagina.
meta name="description" content=""
esta etiqueta es para describir la pagina, dicha descripcion no debe llevar mas de 165 caracteres.
Meta name="autor" content=""
esta etiqueta lleva el nombre del autor de la pagina
meta name="copyright" content=""
esta etiqueta es en caso que la empresa este registrada y tenga copyright, para informar si tiene o no. osea en content va el nombre de la empresa registrada dueña de los derechos de esa empresa.
meta name="robots" content=""
esta se usa para incicar si la pagina es indexable(index) o no(noindex), cuando es index: los buscadores pueden encontrar la pagina facilmente y cuando es noindex: se usa para las paginas que queramos que sean ocultas, osea cuando coloquen una palabra clave en los buscadores referentes a nuestra pagina, esta no aparesca en los buscadores.




HTML semantico

escribir html semantico: es la forma de escribir html para que sea mas legible, mejor y mas optimo para los buscadores.
lo primero que debe tener una pagina web es un encabezado y una barra de navegacion, pueden estar juntos o separados. el header es el encabezado, el encabezado es como la presentacion, de que trata la pagina.
despues tenemos un conetido principal, se utiliza algo llamado articulo(article), este se puede dividir en secciones(section), pueden ser muchos articulos o un solo articulo.
luego tenemos la barra lateral, que es donde va como un articulo segundario, es donde va informacion que no es tan relevante, esta etiqueta se llama (aside), es como la barra lateral.
y por ultimo tenemos el pie de pagina (footer), aca es donde va el copyright,LO QUE NOS MANDA AL ENLACE, lo que nos lleva a la parte de contactos, redes sociales u otras cosas.
